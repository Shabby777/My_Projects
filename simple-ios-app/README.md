# Simple iOS App (SwiftUI)

This folder contains a basic SwiftUI iOS app starter.

## Files
- `SimpleIOSApp/SimpleIOSAppApp.swift` — app entry point
- `SimpleIOSApp/ContentView.swift` — main screen UI with a simple counter

## Run in Xcode
1. Open Xcode.
2. Create a new **iOS App** project named `SimpleIOSApp`.
3. Replace the generated `SimpleIOSAppApp.swift` and `ContentView.swift` with the files from this folder.
4. Choose an iOS simulator and run.
